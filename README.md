## 项目简介

​	本项目根据英语语料库获取了英语语言当中部分高频且具有实际意义的英语单词，并根据这些单词的特征信息，通过计算比较单词与单词之间的余弦相似度，对单词进行分类或聚类的操作。最终使用测试数据集合进行验证。
